# The Aura Inn - Smart Luxury Hotel System

A premium hotel management and booking system featuring:
- **Smart Booking Logic**: 20% Direct Booking Discount.
- **AI Concierge Integration**: WhatsApp-based guest interaction.
- **Admin Dashboard**: Real-time analytics and inventory management.
- **Premium UI**: Dark mode luxury aesthetic with Tailwind CSS.

## Features
- Real-time room availability.
- Automatic email confirmations.
- WhatsApp automation for reminders.
- Revenue tracking & analytics.
- Fully responsive design.

## Tech Stack
- Frontend: EJS, Tailwind CSS, JavaScript.
- Backend: Node.js, Express.
- Database: JSON-based Flat File System.
